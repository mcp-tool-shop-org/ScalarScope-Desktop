
ScalarScope v2 — UI Reference Pack
================================

Purpose
-------
This pack exists to STOP UI drift.

These images are NOT inspiration or styling suggestions.
They are structural targets that define:
- what must be visible
- what must never be hidden
- how the app should orient the user

Contents
--------
01_home_workspace.png
  Target structure for Home screen.
  Replace landing/hero layouts with this.

02_navigation_modes.png
  All modes visible at all times.
  Disabled states explain why.

03_compare_context_header.png
  Mandatory context strip for comparisons.
  Users must always know what they are comparing.

04_delta_zone_hierarchy.png
  Correct visual hierarchy for deltas.
  Evidence-first, not alert-first.

05_guide_lens.png
  Guide as a contextual lens, not a glossary.

Rules
-----
- Do not embellish these layouts.
- Do not add onboarding, tours, or extra copy.
- If a screen does not map to one of these images, reconsider it.

This pack is the final visual authority for v2.
